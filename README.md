# How to use

## 1. Clone Project into your local machine
```
git clone https://github.com/SerhiiNikif/recipe-list.git
```

## 2. Go into project folder

```
cd recipe-list
```

## 3. Start project

```
npm install
```

Terminal 1
```
cd client
```
```
npm run dev
```

Terminal 2
```
cd server
```
```
npm run dev
```

## 4. Play with app now !
Now, you are ready to test app.
Just simply open your browser http://localhost:3000/.
